# 🎯 NumWhisper

**NumWhisper** یک بازی ساده و شیک حدس عدد است که با HTML، CSS و JavaScript خالص ساخته شده. یک عدد تصادفی بین ۱ تا ۵ مخفیانه انتخاب می‌شود — آیا می‌توانی آن را حدس بزنی؟

---

## ✨ ویژگی‌ها

- 🎲 تولید عدد تصادفی بین ۱ تا ۵
- 🪟 طراحی Glassmorphism با پس‌زمینه گرادیان بنفش
- 📱 طراحی واکنش‌گرا برای موبایل و دسکتاپ
- ⚡ بدون هیچ وابستگی خارجی — جاوااسکریپت خالص
- 🎨 انیمیشن‌های روان برای hover و focus

## 🚀 شروع به کار

هیچ نصب یا build ای لازم نیست. کافیه فایل رو در مرورگرت باز کنی:

```bash
git clone https://github.com/your-username/numwhisper.git
cd numwhisper
open index.html
```

یا فقط روی `index.html` دابل‌کلیک کن.

## 🕹️ نحوه بازی

1. روی **Generate Number** کلیک کن — یک عدد مخفی بین ۱ تا ۵ انتخاب می‌شه.
2. حدست رو در فیلد ورودی وارد کن.
3. روی **Submit Number** کلیک کن.
4. 🎉 درست حدس زدی؟ آفرین! ❌ اشتباه؟ عدد درست نمایش داده می‌شه.

## 🗂️ ساختار پروژه

```
numwhisper/
├── index.html    # ساختار اصلی HTML
├── style.css     # استایل Glassmorphism
└── code.js       # منطق بازی
```

## 🛠️ ساخته شده با

- HTML5
- CSS3 (طراحی glassmorphism سفارشی)
- Vanilla JavaScript (ES6)
