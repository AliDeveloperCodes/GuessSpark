# 🎯 NumWhisper

**NumWhisper** is a simple yet stylish number guessing game built with pure HTML, CSS, and JavaScript. A random number between 1 and 5 is secretly generated — can you whisper the right one?

---

## ✨ Features

- 🎲 Generates a random number between 1 and 5
- 🪟 Glassmorphism UI with a smooth purple gradient background
- 📱 Fully responsive design for mobile and desktop
- ⚡ Zero dependencies — pure vanilla JavaScript
- 🎨 Smooth hover and focus animations

## 🚀 Getting Started

No installation or build step required. Just open the file in your browser:

```bash
git clone https://github.com/your-username/numwhisper.git
cd numwhisper
open index.html
```

Or simply double-click `index.html`.

## 🕹️ How to Play

1. Click **Generate Number** — a secret number between 1 and 5 is chosen.
2. Type your guess in the input field.
3. Click **Submit Number**.
4. 🎉 You guessed it? Congrats! ❌ Wrong? The correct number is revealed.
