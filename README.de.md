# 🎯 NumWhisper

**NumWhisper** ist ein einfaches, aber stilvolles Zahlenratespiel, das mit reinem HTML, CSS und JavaScript entwickelt wurde. Eine zufällige Zahl zwischen 1 und 5 wird heimlich ausgewählt — kannst du sie erraten?

---

## ✨ Features

- 🎲 Generiert eine zufällige Zahl zwischen 1 und 5
- 🪟 Glassmorphism-Design mit einem sanften lila Verlaufshintergrund
- 📱 Vollständig responsives Design für Mobilgeräte und Desktop
- ⚡ Keine Abhängigkeiten — reines Vanilla JavaScript
- 🎨 Flüssige Hover- und Fokus-Animationen

## 🚀 Erste Schritte

Keine Installation oder Build-Schritt erforderlich. Einfach die Datei im Browser öffnen:

```bash
git clone https://github.com/your-username/numwhisper.git
cd numwhisper
open index.html
```

Oder einfach auf `index.html` doppelklicken.

## 🕹️ Spielanleitung

1. Klicke auf **Generate Number** — eine geheime Zahl zwischen 1 und 5 wird gewählt.
2. Gib deinen Tipp in das Eingabefeld ein.
3. Klicke auf **Submit Number**.
4. 🎉 Richtig geraten? Glückwunsch! ❌ Falsch? Die richtige Zahl wird aufgedeckt.
